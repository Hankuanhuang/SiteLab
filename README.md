# Site Layout Builder

A web-based site planning and conceptual design tool built for architects, architecture students, and urban design projects.

The application allows users to import site plans, define boundaries, add existing conditions, and create conceptual layouts directly on top of a PDF-based site drawing.

---

## Features

### Site Setup

- Upload PDF Site Plan
- Crop Site Image
- Define Site Boundary
  - Rectangle Mode
  - Polygon Mode
- Set Real-World Dimensions
- Existing Building Mapping
- Existing Tree Mapping
- Green Park Area Mapping
- Road Mapping
- Ancillary Building Mapping

### Site Editor

- Interactive Site Layout Design
- Zoom In / Zoom Out
- Pan Canvas
- Fit To Screen
- Distance Measurement Tools
- Building Placement
- Entrance Placement
- Sidewalk Creation
- Bridge Tool
- Core Tool
- Toilet Tool
- Object Rotation
- Copy / Paste Objects
- Undo / Redo

### Export

- Export Concept Site Plan
- Save Layout
- Project Gallery

## Project Goal

Site Layout Builder was created to simplify the early-stage site planning process.

Instead of drawing conceptual layouts manually, users can quickly:

1. Import a site plan
2. Define site conditions
3. Create building layouts
4. Test design ideas
5. Export concept plans

The project combines architecture and software development to improve design workflows.
